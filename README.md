## 显示 ssh .config 内容

读取 `~/.ssh/config` 中的 `Host` 部分内容，支持选择某个 host 后直接打开 iTerm2。

在 hosts 列表中支持执行 Alfred 默认操作：

1. ⌘ + C：拷贝选中的 host
2. ⌘ + L：放大显示选中行内容

![example](./ssh//example.png)
